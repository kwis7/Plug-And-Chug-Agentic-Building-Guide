# Facilitation Script

Use this when helping a non-technical person build their first system.

## Opening

"We are not trying to make AI smarter by adding more prompts. We are building a small local filing system so AI knows where to look, what role it is playing, and where to write down useful state when the conversation ends."

## Three Questions

1. "What are the recurring areas where you ask AI for help?"
2. "Which of those areas contain private raw files?"
3. "Which tasks repeat often enough that a checklist would save effort?"

## Gentle Corrections

If the user wants one huge agent:

"That works at first, but it gets messy. Let's split by life domain so job materials do not mix with research notes or family paperwork."

If the user wants many tiny agents:

"Let's start with fewer rooms. A repeated task can become a skill inside an agent before it becomes a new agent."

If the user wants memory to store everything:

"Memory should be compact. Raw files stay in raw folders; memory stores the pointer and the decision."

## Closing

"Your next test is simple: after one real task, can tomorrow's AI session read the folder and know what happened, what matters, and what to do next?"
