# Mental Model

## The Problem

Many people have dozens of useful AI conversations but no durable system. The AI gives advice, writes a draft, finds a source, or plans a task. Then the next conversation starts from zero.

## The System

Think of the system as a small office:

- the front desk is the control center;
- each recurring area has its own room;
- each room has rules, a notebook, a library shelf, source cabinets, and recipe cards;
- a status board shows what is active, blocked, done, or stale;
- each active task has a small task card;
- sensitive source files stay in locked cabinets;
- when work ends, reviewed deliverables go to the output tray and the assistant writes only the compact notes needed for next time.

## The Payoff

The goal is not more prompts. The goal is less chaos:

1. fewer repeated explanations;
2. clearer privacy boundaries;
3. reusable workflows;
4. easier handoff between AI tools;
5. local files that humans can inspect.

## The Smallest Useful Version

Everything else can grow later.
