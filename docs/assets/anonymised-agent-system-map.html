How To Read This

  • The root routes work and keeps the system map; it does not do every task itself.
  • Main agents keep recurring context separate, so memory does not become soup.
  • Subagents isolate long, sensitive, or high-context projects.
  • By default, only reviewed files from outputs/ are ready to share.
